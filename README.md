# Bankruptcy Prediction using Machine Learning

## Project Overview
This project predicts whether a company will go bankrupt using financial indicators and machine learning algorithms.

## Problem Statement
Financial institutions need to identify companies at risk of bankruptcy to minimize financial losses.

## Dataset
The dataset contains financial ratios and indicators related to company performance.

## Technologies Used
Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Scikit-learn  

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest

## Project Workflow
1. Data Preprocessing
2. Exploratory Data Analysis
3. Feature Selection
4. Model Training
5. Model Evaluation

## Results
Random Forest achieved the best prediction performance.
